# Translations directory
Add your translation into your `translations/mod-id/` directory.
```
[translations]
    [my-mod-id]
        [EN]
            Sandbox_EN.txt
            Moodles_EN.txt

        [IT]
            Sandbox_IT.txt
            Moodles_IT.txt

    [my-other-mod-id]
        [EN]
            Farming_EN.txt
            IG_UI_EN.txt

        [IT]
            Farming_IT.txt
            IG_UI_IT.txt
```
